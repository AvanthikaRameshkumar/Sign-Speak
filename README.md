# Sign-Speak

## Description :
 SignSpeak is a system that translates speech or text into sign language gestures, helping improve communication for hearing and speech-impaired individuals. It focuses on accessibility by converting input into understandable gestures, bridging the gap between sign language users and others.
## Features :
 Converts speech into sign language gestures  - Converts text into sign language gestures  - Improves communication for hearing and speech-impaired individuals  - Simple and user-friendly interface  - Helps bridge communication gap between users.
## Technologies Used :
 Python 3.x  - Flask  - OpenAI API  - HTML, CSS, JavaScript.
